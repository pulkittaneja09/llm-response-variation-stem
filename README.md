# Prompt Sensitivity and Response Variation in Large Language Models for STEM Education

This repository contains the dataset, analysis code, and figures used in the research project:

**‚ÄúPrompt Sensitivity and Response Variation in Large Language Models for STEM Education.‚Äù**

The study investigates how different prompt phrasing styles influence the **complexity**, **sentiment**, and **behavioral stability** of two widely used large language models ‚Äî **ChatGPT** and **Gemini** ‚Äî when answering STEM-based questions.

---

## üìÇ Dataset Overview

The dataset consists of **240 prompt‚Äìresponse pairs**, generated by combining:
- **30 STEM questions**
- **4 prompt phrasing types**  
  - Neutral  
  - Leading  
  - Contradictory  
  - Self-Doubt  
- **2 LLMs** (ChatGPT, Gemini)

Each response is annotated with:
- **ACCURACY** (Correct / Incorrect)  
- **COMPLEXITY** (1‚Äì5 scale)  
- **SENTIMENT** (Positive / Neutral / Negative)

### File:
`llm_prompt_sensitivity_dataset.csv`

### Columns:
| Column Name   | Description |
|---------------|-------------|
| QID           | Unique question ID |
| QUESTION      | Original STEM question |
| PROMPT TYPE   | Neutral / Leading / Contradictory / Self-Doubt |
| MODEL         | ChatGPT or Gemini |
| RESPONSE      | Model-generated answer |
| ACCURACY      | Correct or Incorrect |
| COMPLEXITY    | Response complexity (1‚Äì5) |
| SENTIMENT     | Positive / Neutral / Negative |

## üìà Statistical Analyses

The analysis includes:

- **Chi-Square Test**  
  - Tests association between Prompt Type and Sentiment

- **Data visualizations**
